# 概述

# 平台定位

平台共项目管理、资源管理、组件基座三大主体，项目管理包括：项目管理、资源管理、运行状态、开发工作台，资源管理包括:项目资源、平台资源。各类型组件与资源是平台的基础，工作台是平台的主体，资源管理库是平台的中枢。平台是建立在各类组件及资源套件之上，快速将各类页面需求转换为页面应用。

通过可视化进行应用程序开发的方法，使具有不同经验水平的开发人员可以通过图形化的用户界面开箱即用，快速投入，在使用拖拽或标签布局等规范化方法组合各类组件与模型逻辑创建桌面应用。

# 术语与定义

# BI

商业智能（Business Intelligence，简称：BI），又称商业智慧或商务智能，指用现代数据仓库技术、线上分析处理技术、数据挖掘和数据展现技术进行数据分析以实现商业价值。

# IV

信息可视化（Information visualization）是一个跨学科领域，旨在研究大规模非数值型信息资源的视觉呈现。通过利用图形图像方面的技术与方法，帮助人们理解和分析数据。

# UI编辑

UI可视化编辑器是一款通过UI组件拖拽生成前端页面的编辑器系统，用户可以通过丰富的组件属性来自定义组件样式和功能，从而快速开发可视化页面。

# 低代码

低代码开发平台（LCDP）是无需编码（0代码）或通过少量代码就可以快速生成应用程序的开发平台。通过可视化进行应用程序开发的方法，使具有不同经验水平的开发人员可以通过图形化的用户界面，使用拖拽组件和模型驱动的逻辑来创建网页和移动应用程序。

# 标准化接口模型

采用中立的方式进行定义，独立于实现服务的硬件系统、操作系统和编程语言。它将应用程序的不同功能划分为服务，通过这些服务之间定义良好的契约联系起来。基于该接口构建的各种系统中的服务可以以一种统一和通用的方式进行交互。
